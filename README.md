O que são dados?
Dados são valores brutos que podem ser transformados em informações
O que é informações?
Informações são dados lapidados sobre uma determinada coisa
O que é um banco de dados?
É um lugar organizado e seguro para guardas informações
Analogia
Dados:25,60kg
Informações:25 anos de idade,60kg é o peso de alguém
Banco de dados: Cadastro do posto de saúde que vai guardar o nome e o peso de alguém
